There are more units for describing size other than Pixels. Like : rem, em, vw, vh, %

What's wrong with Pixels ?
            ans :- Pixels are relative to the viewing device. So, that's why it can vary with different-different devices.

em :: Related to it's parent font-size. It can compound along with the parent font-size.

rem :: is always releated to HTML ROOT font-size. which is 16px by-default.

vw :: Unit relative to 1% of viewport width.

vh :: Unit relative to 1% of viewport height.

% :: Unit relative to the parent element.
