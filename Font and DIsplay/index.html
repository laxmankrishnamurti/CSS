Font and Display

The CSS display property is used to determine whether an element is treated as a "Block" or "Inline" element.

Inline :: Takes only the space required bu the element. No line break before and after. Setting widht and height is not allowed.

Block :: Takes full available in widht and leaves a newline before and after the element.

Inline-block :: It is quite similar to inline but setting widht and height is allowed. Element can sit next to each other.
